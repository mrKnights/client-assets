# Client Assets

A centralized public repository for hosting static assets used across client websites and digital projects managed by **MR KNIGHT'S**.

## Overview

This repository contains publicly accessible static resources such as logos, images, icons, banners, illustrations, and other media required for websites, web applications, landing pages, and digital platforms.

The primary purpose of this repository is to provide a reliable, version-controlled, and CDN-compatible source for static assets that can be referenced directly from client projects.

## Guidelines

* Organize assets inside the corresponding client directory.
* Use descriptive and consistent filenames.
* Prefer optimized formats such as SVG and WebP whenever possible.
* Keep unnecessary duplicates to a minimum.
* Do not upload confidential, private, or licensed files unless they are approved for public distribution.

## Version Control

All assets are maintained using Git version control, allowing updates, revisions, and historical tracking while providing stable references for production environments.
